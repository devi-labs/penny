# HTML Basics

## Topic
HTML as the skeleton of a web page. Cover tags, headings (h1-h3), paragraphs (p), links (a), images (img). Show a minimal page structure.

## Challenge
Write a simple HTML page with your name as a heading and a short bio in a paragraph.

## Success Criteria
Has at least one heading tag (h1, h2, or h3) and one paragraph tag (p). Contains some text content. Doesn't need to be a full valid HTML document — just the tags.
