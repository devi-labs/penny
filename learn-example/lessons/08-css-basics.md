# CSS Basics

## Topic
CSS makes HTML look good. Cover selectors, colors, fonts, spacing (margin/padding), background colors. Style the page from the previous lesson.

## Challenge
Write CSS that changes the background color and text color of a page. Style a heading to be a different font size.

## Success Criteria
Has at least one CSS rule with a selector and curly braces. Sets at least one visual property (color, background-color, font-size, etc.). Valid CSS syntax.
