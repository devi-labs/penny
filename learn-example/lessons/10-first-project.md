# Build Your Homepage

## Topic
Put everything together — build a personal homepage using HTML, CSS, and JavaScript. Guide them through creating a page with their name, bio, favorite things, and a button that does something. This is their first real project.

## Challenge
Build a personal homepage with: a heading with your name, a paragraph about yourself, at least one styled element (CSS), and one interactive thing (JavaScript). Send the full code!

## Success Criteria
Has HTML structure with heading and paragraph. Has some CSS (inline or in a style tag). Has some JavaScript (inline or in a script tag). Combines all three technologies from the previous lessons. Doesn't need to be perfect — any reasonable attempt that uses HTML + CSS + JS together counts.
