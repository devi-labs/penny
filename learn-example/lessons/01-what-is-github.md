# What is GitHub?

## Topic
Explain GitHub to a total beginner. Compare it to Google Docs for code. Cover: repos, files, commits. Mention the GitHub mobile app.

## Challenge
Open the GitHub app on your phone and find any repo. Tell me the name of one file you see.

## Success Criteria
Any filename-like response (e.g. README.md, index.html). Just needs to show they looked at a repo.
